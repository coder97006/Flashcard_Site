Concepts You Should Know:

  • JavaScript Definition: programming language used for logic in webpages.
  • Operators Definition: Addition, Subtraction, Multiplication, Division etc.
  • Variables Definition: Const, Let and Var: what holds or is assigned a value
  • Functions Definition: aka method or a block of code that performs a specific task when called
  • Methods Definition: a function that is available to specific objects
  • jQuery Definition: A javascript library that has pre made modules for common tasks
  • Booleans Definition: Logic operators that are used in programming languages
